SLO-Aware Multi-Agent LLM Serving (Work in Progress)

This ongoing project explores how to efficiently serve multi-agent large language model applications under latency, throughput, and energy constraints. Multi-agent LLM systems often consist of multiple specialized agents, such as planners, actors, retrievers, and synthesizers. These agents interact through a task graph, making the serving process more complex than single-model inference.

We investigate SLO-aware routing, dynamic model selection, and hardware-aware scheduling for multi-agent LLM workloads. The goal is to improve end-to-end response latency and serving efficiency while satisfying request-level service objectives.

Extended version in preparation for conference submission.

Benchmarking Prefill-Decode Disaggregation for LLM Inference

Large language model inference consists of two major stages: prefill and decode. Prefill processes input prompts and builds the KV cache, while decode generates output tokens autoregressively. These two stages exhibit different compute, memory, and energy characteristics.

This project benchmarks prefill-decode disaggregation across different model sizes, workload settings, KV cache transfer paths, and GPU frequency configurations. We study when disaggregation improves performance and energy efficiency, and when its overheads outweigh the benefits.

The project also explores SLO-aware serving-efficiency metrics that combine token throughput, energy consumption, and latency constraints. The goal is to provide practical guidance for deploying disaggregated LLM inference systems.

Efficient Video Analytics Systems for Edge and IoT Applications

Video analytics systems are increasingly deployed on edge devices, embedded platforms, and IoT environments. These systems must satisfy strict latency and resource constraints while processing high-volume visual data streams.

This project studies efficient video analytics pipelines for real-world deployment scenarios. We investigate model selection, adaptive scheduling, and resource-aware optimization to improve system performance under dynamic workloads.

The broader goal is to design practical AI systems that bridge machine learning accuracy with deployment constraints such as compute capacity, bandwidth, energy consumption, and real-time responsiveness.
